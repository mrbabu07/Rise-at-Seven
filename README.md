# Rise-at-Seven

A pixel-perfect clone of the Rise at Seven agency website built with modern web technologies.

## 🚀 Tech Stack

- **Frontend**: React 19, Vite 8
- **Styling**: Tailwind CSS 4
- **Animations**: GSAP 3.13, Lenis
- **Interactive**: Alpine.js, Swiper
- **Transitions**: Barba.js

## 📱 Mobile Responsive

This project features pixel-perfect mobile responsiveness matching the original Rise at Seven website:

- **Mobile**: ≤ 620px (iPhone, Android phones)
- **Tablet**: 621px - 980px (iPad, tablets)  
- **Desktop**: > 980px (laptops, desktops)

## 🛠️ Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
